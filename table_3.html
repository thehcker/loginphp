Course structure

 duration
Section One One Week
Rationale of data analysis using R 1 day
Defining objects and performing basic arithmetic & logical operations 1 day
Use predefined functions to perform operations on objects 2 days
Distinguishing between various data types 2 days
Section Two One Week
Creating numeric and character vectors
Naming columns of a vector
Generating numeric sequences
Accessing specific elements or parts of a vector
Coercing data into different data types as needed
Sort vectors in ascending and descending order
Extract the indices of the sorted elements from the original vector
Find the max and min elements as well as their indices in a vector
Rank the elements of a vector in increasing order
Perform arithmetic between a vector and a single number
Perform arithmetic between two vectors of the same length
Section Three One Week
Subset a vector based on properties of another vector 1 day
Use multiple logical operators to index vectors 1 day
Extract the indices of vector elements satisfying one or more logical conditions 1 day
Extract the indices of vector elements matching with another vector 1 day
Determine which elements in one vector are present in another vector 1 day
Section Four One Week
Conditional expressions e.g For loops and Functions 1 day
Basic conditionals e.g if-else and ifelse 1 day
Other functions e.g apply, sapply, tapply,mapply, split,cut, quantile,reduce,identical,unique 2 days
Reporting hierarchical data 2 days
